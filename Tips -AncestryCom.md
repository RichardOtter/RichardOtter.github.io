Last Update: 2026-03-15

## URL general format

Tree URL
https://www.ancestry.com/family-tree/tree/14741034/

Tree Overview
https://www.ancestry.com/family-tree/tree/14741034/recent

Tree Settings
https://www.ancestry.com/family-tree/tree/14741034/settings/info

Tree URL with tree centered at person with ID: 155162780
https://www.ancestry.com/family-tree/tree/14741034/family?cfpid=155162780

Profile URL
https://www.ancestry.com/family-tree/person/tree/14741034/person/155162780/facts

DNA Match list
https://www.ancestry.com/discoveryui-matches/list/3472852C-4089-4840-84A0-682AC8631A96

DNA Match
https://www.ancestry.com/discoveryui-matches/compare/3472852C-4089-4840-84A0-682AC8631A96/with/2728142C-B35D-4A2E-B2A3-6C3E3B80C358

People List
https://www.ancestry.com/family-tree/tree/14741034/listofallpeople


List of owned trees
https://www.ancestry.com/family-tree/tree

## Hint List for a particular collection

Hint list for a particular source collection for a particular tree

for example, the source collection:
California Birth Index, 1905-1995
https://www.ancestry.com/search/collections/5247/
Collection ID= 5247

For my main tree on Ancestry
https://www.ancestry.com/family-tree/tree/14741034/
Tree ID= 14741034

Page listing all hints currently established in my tree for links to the
California Birth Index:
https://www.ancestry.com/hints/tree/14741034/hints?hf=record&hs=last&hdbid=5247


in general:
https://www.ancestry.com/hints/tree/<treeID>/hints?hf=record&hs=last&hdbid=<collectionID>


I create a bookmark for each of the collections I'm particularly interested in,
and I put them in the Chrome bookmark list in a subfolder within Chrome's
Bookmark manager.
