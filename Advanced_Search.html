FIELD 1

Special Search Items

Field one selection menu contains an entry for- each standard fact, each custom fact, and the special search items listed here:
  • Given name(s)
  • Given or Nickname
  • Surname
  • Surname (Birth or Marr)
  • Married surname
  • Nickname
  • Prefix
  • Suffix
  • Sex
  • Living flag
  • Date edited
  • Current age
  • Record number
  • Any Fact
  • Color coding
  • FamilySearch ID
  • Note (family)
  • Note (general)
  • Number of children
  • Number of sets of parents
  • Number of siblings
  • Number of spouses
  • Source (family)
  • Source (general)
Unfortunately, these search items are mixed in with the large list of Fact names. Some users never find them.
